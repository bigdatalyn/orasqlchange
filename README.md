## OSC
Convert heterogeneous SQL with clear Oracle targets.
Ora SQL Change converts MySQL, SQL Server, PostgreSQL, and Db2 DDL, DML, queries, and selected stored programs for Oracle 12c, 19c, and AI Database 26ai.
